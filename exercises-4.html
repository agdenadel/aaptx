2

For each of the following functions, prove or disprove that the function is (i) a homomorphism; (ii) an isomorphism. (Remember to work with the default operation on each of these groups!)

  1. The function \(f:\Z\to\Z\) defined by \(f(n)=2n\text{.}\)

  2. The function \(g:\R\to\R\) defined by \(g(x)=x^2\text{.}\)

  3. The function \(h:\Q^*\to\Q^*\) defined by \(h(x)=x^2\text{.}\)

Solution

    1. The function \(f\) is a homomorphism, since for every \(a,b \in \Z,\) we have

      \begin{equation*} f(a+b)=2(a+b)=2a+2b=f(a)+f(b). \end{equation*}

    2. The function \(f\) isn't a bijection, since it isn't onto: for instance, there is no element \(a\) in \(\Z\) such that \(f(a)=3\text{.}\) Thus, \(f\) is not an isomorphism.

    1. The function \(g\) isn't a homomorphism: for instance, we have

      \begin{equation*} g(2+3)=g(5)=25\neq 13=4+9=g(2)+g(3). \end{equation*}

    2. Since it isn't a homomorphism, it isn't an isomorphism.

    1. The function \(h\) is a homomorphism, since for every \(a,b \in \Q,\) we have

      \begin{equation*} h(ab)=(ab)^2=a^2b^2=h(a)h(b). \end{equation*}

    2. The function \(h\) isn't onto: its range is the set of nonnegative rational numbers, not all of \(\Q\text{.}\) Thus, \(h\) is not an isomorphism.
3

Define \(d : GL(n,\R)\to \R^*\) by \(d(A)=\det A\text{.}\) Prove/disprove that \(d\) is:

  1. a homomorphism

  2. 1-1

  3. onto

  4. an isomorphism.

Solution

  1. The function \(d\) is a homomorphism, since for every \(A,B \in GL(n,\R),\) we have

    \begin{equation*} d(AB)=\det(AB)=(\det A)(\det B)=d(A)d(B). \end{equation*}

  2. \(d\) isn't 1-1: For instance, \(d(I_2)=d(-I_2)\text{.}\)

  3. \(d\) is onto: Let \(a\in \R^*\text{.}\) Then the matrix

    \begin{equation*} A:=\begin{bmatrix}a \amp 0 \amp 0 \amp \cdots \amp 0 \\ 0 \amp 1 \amp 0 \amp \cdots \amp 0 \\ 0 \amp 0 \amp 1 \amp \cdots \amp 0 \\ \vdots \amp \vdots \amp \vdots \amp \ddots \amp \vdots \\ 0 \amp 0 \amp 0 \amp \cdots \amp 1 \end{bmatrix} \end{equation*}

    is in \(GL(n,\R)\text{,}\) with \(d(A)=a\text{.}\)

  4. Since \(d\) isn't 1-1, it isn't an isomorphism.

5

Let \(n\in \Z^+\text{.}\) Prove that \(\langle n\Z,+\rangle \simeq \langle \Z,+\rangle\text{.}\)

Solution

Define \(\phi\,:\,\Z \rightarrow n\Z\) by \(\phi(x)=nx\text{,}\) for all \(x\in \Z\text{.}\) Clearly, \(\phi\) is a bijection. Moreover, for all \(x,y\in \Z\text{,}\)

\begin{equation*} \phi(x+y)=n(x+y)=nx+ny=\phi(x)+\phi(y); \end{equation*}

so \(\phi\) is a homomorphism. Thus, \(\phi\) is an isomorphism, and hence

\begin{equation*} \langle \Z, +\rangle \simeq \langle n\Z, +\rangle . \end{equation*}
6

  1. Let \(G\) and \(G'\) be groups, where \(G\) is abelian and \(G\simeq G'\text{.}\) Prove that \(G'\) is abelian.

  2. Give an example of groups \(G\) and \(G'\text{,}\) where \(G\) is abelian and there exists a homomorphism from \(G\) to \(G'\text{,}\) but \(G'\) is NOT abelian.

Solution

  1. Since \(G\simeq G'\text{,}\) there exists some isomorphism \(\phi:G\to G'\text{.}\) Let \(x,y\in G'\text{.}\) Since \(\phi\) is onto, there exist \(a,b\in G\) such that \(\phi(a)=x\) and \(\phi(b)=y\text{.}\) So

    \begin{align*} xy\amp =\phi(a)\phi(b)\amp \amp\\ \amp =\phi(ab)\amp \amp \text{ (since \(\phi\) is a homomorphism) }\\ \amp =\phi(ba) \amp \amp \text{ (since \(G\) is abelian) }\\ \amp =\phi(b)\phi(a)\\ \amp =yx. \end{align*}

    Thus, \(G'\) is abelian.

  2. Let \(G=\{I_2\}\) (under multiplication) and let \(G'=GL(2,\R)\text{.}\) Then the map \(\phi: G\to G'\) defined by \(\phi(I_2)=I_2\) is clearly a homomorphism, but note that \(G\) is abelian while \(G'\) is not.
7

Let \(\langle G,\cdot\rangle\) and \(\langle G',\cdot'\rangle\) be groups with identity elements \(e\) and \(e'\text{,}\) respectively, and let \(\phi\) be a homomorphism from \(G\) to \(G'\text{.}\) Let \(a\in G\text{.}\) Prove that \(\phi(a)^{-1}=\phi(a^{-1})\text{.}\)

Solution
Note3.4.20

We omit the group operations in this solution in order to increase familiarity with the convention.

We want to show that \(\phi(a)^{-1}=\phi(a^{-1})\text{.}\) Notice that

\begin{align*} \phi(a^{-1})\phi(a)\amp =\phi(a^{-1}a)\amp \amp \text{ (since \(\phi\) is a homomorphism) }\\ \amp =\phi(e)\amp \amp \text{(by definition of \(a^{-1}\))}\\ \amp =e'\amp \amp \text{ (since \(\phi(e)\) is the identity element of \(G'\)) }\\ \amp =\phi(a)^{-1}\phi(a)\amp \amp \text{(by definition of \(\phi(a)^{-1}\))} . \end{align*}

Thus, \(\phi(a^{-1})=\phi(a)^{-1}\text{,}\) by right cancellation.
