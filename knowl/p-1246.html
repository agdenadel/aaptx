We also have that \(\Phi\) is onto, since if \(r\in \R\text{,}\) then the constant function \(c_r\) defined by

\begin{equation*} c_r(x)=r \mbox{ for every \(x\in [0,1]\)} \end{equation*}

is sent to \(r\) by \(\Phi\text{.}\) So \(\Phi(F)=\R\text{.}\) Finally, if \(f\in F\text{,}\) then

\begin{equation*} f\in \Ker \Phi \Leftrightarrow \Phi(f)=0 \Leftrightarrow f\left(\frac{1}{4}\right)=0 \Leftrightarrow f\in N; \end{equation*}

so \(\Ker \Phi=N\text{.}\) Thus, \(F/N \cong \R\text{,}\) by the First Isomorphism Theorem.
