Note that if \(\sigma\) in \(S_n\) is written in cycle notation and the number \(a\in \{1,2,\ldots, n\}\) appears nowhere in \(\sigma\)'s representation, this means that \(\sigma\) fixes \(a\text{.}\) The only permutation that we cannot really write in cycle notation is the identity element \(1_A\) of \(S_A\text{,}\) which we henceforth denote by \(e\text{.}\)
