Example6.1.2

Let \(A\) be the set \(A=\{\Delta, \star, 4\}\text{.}\) Then the functions \(\sigma : A\to A\) defined by \begin{equation*} \sigma(\Delta)=\star, \sigma(\star)=\Delta, \text{and } \sigma(4)=4; \end{equation*} and \(\tau : A\to A\) defined by \begin{equation*} \tau(\Delta)=4, \tau(\star)=\Delta, \text{and } \tau(4)=\star \end{equation*} are both permutations on \(A\text{.}\)
