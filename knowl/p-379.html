Let's start exploring groups in order of increasing, well, order. Before we do this, it will be helpful to introduce the notion of a group table (also known as a Cayley table 1 Named after the British mathematician Arthur Cayley.) for a finite group. Given a finite group \(G\text{,}\), list its elements in some fixed order, say, \(a_1, a_2, \ldots, a_n\text{,}\), and then construct its group table by creating an array with exactly one row and exactly one column corresponding to each group element. We then put in the row \(i\) and column \(j\) the element \(a_ia_j\) of \(G\text{.}\). Note that a single group can have group tables that look different from one another, since reordering a group's elements will change its table.
