Solution3.4.3.1

  1. The function \(d\) is a homomorphism, since for every \(A,B \in GL(n,\R),\) we have \begin{equation*} d(AB)=\det(AB)=(\det A)(\det B)=d(A)d(B). \end{equation*}

  2. \(d\) isn't 1-1: For instance, \(d(I_2)=d(-I_2)\text{.}\)

  3. \(d\) is onto: Let \(a\in \R^*\text{.}\) Then the matrix \begin{equation*} A:=\begin{bmatrix}a \amp 0 \amp 0 \amp \cdots \amp 0 \\ 0 \amp 1 \amp 0 \amp \cdots \amp 0 \\ 0 \amp 0 \amp 1 \amp \cdots \amp 0 \\ \vdots \amp \vdots \amp \vdots \amp \ddots \amp \vdots \\ 0 \amp 0 \amp 0 \amp \cdots \amp 1 \end{bmatrix} \end{equation*} is in \(GL(n,\R)\text{,}\) with \(d(A)=a\text{.}\)

  4. Since \(d\) isn't 1-1, it isn't an isomorphism.
