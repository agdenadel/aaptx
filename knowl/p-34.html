Sets \(A\) and \(B\) are disjoint if \(A\cap B=\emptyset\text{.}\) More generally, sets \(A_i\) (\(i\) in some index set \(I\)) are disjoint if

\begin{equation*} \bigcap_{i\in I}A_i=\emptyset \end{equation*}

and are mutually disjoint if

\begin{equation*} A_i\cap A_j=\emptyset \text{ for all } i\neq j \in I. \end{equation*} in-context
