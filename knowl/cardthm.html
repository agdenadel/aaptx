Theorem1.3.7

Let \(A\) and \(B\) be sets.

  1. If \(A\subseteq B\) and \(A\) is infinite [uncountable] then so is \(B\text{.}\)

  2. If \(A\subseteq B\) and \(B\) is finite [countable] then so is \(A\text{.}\)

  3. If \(|A|=n\lt \infty\) and \(|B|=m\lt \infty\text{,}\) then \(|A\times B|=mn\text{.}\)

  4. Any finite product of countable sets is countable.
