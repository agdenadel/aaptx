Notice that in additive notation, the statement “\(a^{-1}b\in H\)” becomes \(-a+b\in H\text{.}\). So for \(a,b\in \Z\text{,}\), \(a\siml b\) if and only if \(-a+b \in 5\Z\text{;}\); that is, if and only if \(5\) divides \(b-a\text{.}\). In other words, \(a\siml b\) if and only if \(a\equiv_5 b\text{.}\). So in this case, \(\siml\) is just congruence modulo \(5\text{.}\). Thus, the cosets of \(5\Z\) in \(\Z\) are \begin{align*} 5\Z\amp =\{\ldots,-5,0,5,\ldots\}\\ 1+5\Z\amp =\{\ldots,-4, 1, 6,\ldots\},\\ 2+5\Z\amp =\{\ldots,-3,2, 7, \ldots\},\\ 3+5\Z\amp =\{\ldots,-2,3, 8, \ldots\},\\ 4+5\Z\amp =\{\ldots,-1, 4, 9, \ldots\}. \end{align*}
