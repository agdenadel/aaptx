First, assume \(o(a)=n\lt \infty\text{.}\). Then \begin{equation*} (a^{-1})^n=(a^n)^{-1}=e^{-1}=e, \end{equation*} so \(o(a^{-1})\leq n=o(a)\text{.}\). Using the same argument, we have \(n=o(a)\leq o(a^{-1})\text{.}\). Since \(o(a^{-1})\leq n\) and \(n\leq o(a^{-1})\text{,}\), \(o(a^{-1})=n=o(a)\text{.}\).
