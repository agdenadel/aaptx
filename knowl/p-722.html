Given a set \(A\text{:}\)

  1. \(S_A\) is a group under permutation multiplication.

  2. If \(A\) has finite cardinality \(n\text{,}\) then \(|S_A|=n!\) (if \(|A|=\infty\) then \(|S_A|\) is also \(\infty\)).

  3. \(S_A\) is abelian if \(|A|=1\) or \(2\text{,}\) and nonabelian otherwise.
