A group is a set \(G\text{,}\) equipped with a binary operation \(*\text{,}\) that satisfies the following three group axioms:

  1. \(*\) is associative on \(G\) (we call this axiom \(\G_1\))

  2. There exists an identity element for \(*\) in \(G\) (axiom \(\G_2\))

  3. Every element \(a\in G\) has an inverse in \(G\) (axiom \(\G_3\))
