Definition7.2.17

We let \(G/H\) be the set of all left cosets of subgroup \(H\) in \(G\text{.}\) We read \(G/H\) as “\(G\) mod \(H\text{.}\)” (We denote the set of all right cosets of subgroup \(H\) in \(G\) by \(H\backslash G\text{,}\) but we will not be explicitly referring to that set in this class.)
