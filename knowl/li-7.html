Item2

More generally, given any collection of sets \(A_i\) (\(i\) in some index set \(I\)), the union of the \(A_i\) is

\begin{equation*} \bigcup_{i\in I}A_i=\{x: x\in A_i \text{ for some } i\in I\}, \end{equation*}

and the intersection of the \(A_i\) is

\begin{equation*} \bigcap_{i\in I}A_i=\{x: x\in A_i \text{ for every } i\in I\}. \end{equation*}
