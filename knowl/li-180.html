  • \(d\) is onto: Let \(a\in \R^*\text{.}\) Then the matrix \begin{equation*} A:=\begin{bmatrix}a \amp 0 \amp 0 \amp \cdots \amp 0 \\ 0 \amp 1 \amp 0 \amp \cdots \amp 0 \\ 0 \amp 0 \amp 1 \amp \cdots \amp 0 \\ \vdots \amp \vdots \amp \vdots \amp \ddots \amp \vdots \\ 0 \amp 0 \amp 0 \amp \cdots \amp 1 \end{bmatrix} \end{equation*} is in \(GL(n,\R)\text{,}\) with \(d(A)=a\text{.}\)
