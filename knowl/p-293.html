Finally, let \(a\in \Z_n\text{.}\). If \(a=0\text{,}\), then clearly \(a\) has inverse \(0\in \Z_n\) since \(0+_n 0 = 0\text{.}\). If \(a\neq 0\text{,}\), then the element \(n-a\in \Z_n\) is an inverse for \(a\) since \begin{equation*} a+_n(n-a)=R_n(a+(n-a))=R_n(n)=0. \end{equation*}
