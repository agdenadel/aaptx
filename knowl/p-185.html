A group is a set \(G\text{,}\), equipped with a binary operation \(*\text{,}\), that satisfies the following three group axioms:

  1. \(*\) is associative on \(G\text{.}\) (Axiom \(\G_1\))

  2. There exists an identity element for \(*\) in \(G\text{.}\) (Axiom \(\G_2\))

  3. Every element \(a\in G\) has an inverse in \(G\text{.}\) (Axiom \(\G_3\))
